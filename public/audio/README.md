# Audio Assets

This folder contains ambient audio files for the focus timer.

## Required File

Place a looping ambient rain/forest sound here:
- **`rain.mp3`** — used by the timer page audio toggle

## Where to get it

Free, royalty-free options:
- https://freesound.org (search "rain loop")
- https://pixabay.com/music/ (search "rain ambience")
- https://soundbible.com

Download any rain/nature loop as MP3, rename it `rain.mp3`, and place it in this folder.

## Notes
- Keep file size under 3 MB for fast loading
- Must be a seamless loop (no gap at beginning/end)
- Audio only plays when user explicitly enables it during a session
